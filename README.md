# Order-processing
This is a project to refresh and enhance my knowledge of microservices, spring, kafka, and other concepts. 

I am also working on this using GitHub Copilot to see how I can work with it and accelerate re-acclimating to these technologies.

There will be several microservices for the "order-processing" and I've added some additional services along the way as needed (i.e. palpay).

The services are as follows:
- inventory
- order
- payment (incomplete)
- paypal
- shipping (not started)

The docker compose will bring up these services as well as Kafka and a SQL database. I intend to add keycloak in the future as well.

For the sake of clarity on the use of Copilot, it has done most of the work in test making and helped with additional coding in the service classes, but mostly as refactoring/large code completion.
